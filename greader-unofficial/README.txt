Google Reader Unofficial API (http://code.google.com/p/greader-unofficial/)
------------------------------------
The goal of this project is to provide an unofficial API for Google Reader.


Comments:
No warranty is given that the provided could will remain functional. 
We are totally dependent of the changes made by Google, which we have no control over.
